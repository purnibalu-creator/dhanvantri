# dhanvantri

This app offers traditional wellness guidance rooted in Ayurvedic and homeopathic traditions. It is not a substitute for professional medical advice. Always consult a qualified physician for serious conditions. Users can describe ailments and symptoms to which this app will respond with detailed analysis to explain the root cause of the problem/ ailment/ suffering. The app will provide a detailed understanding of the condition. It'll then recommend herbs and formulations to take as advice providing some easy and classical remedies. It'll also suggest lifestyle changes and dietary adjustments to ensure there is a healing process that can be practiced for bettering the symptoms.
